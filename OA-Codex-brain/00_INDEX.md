# OA Codex Brain Index

This folder is a working memory and execution brain for MStorm Studio 2026.

It is designed to do two jobs:
- Preserve accurate repo understanding as the codebase changes.
- Help a small engineering team execute with speed and low confusion.

## File Map
- `01_REPO_UNDERSTANDING.md`: Architecture, structure, and what exists right now.
- `02_ETHOS_AND_PRODUCT_PRINCIPLES.md`: Product philosophy and non-negotiables.
- `03_ACTIVE_STATE_2026-03-28.md`: Current implementation reality and gaps.
- `04_ACTION_PLAN_AND_ROADMAP.md`: Practical next steps and sequencing.
- `05_MULTI_ENGINEER_EXECUTION_MODEL.md`: Plan for a small team building this repo.
- `06_INTELLIGENT_DEV_METHOD.md`: Process for building smarter, evidence-driven software.
- `07_TEAM_DISCUSSION_AND_ASSESSMENT.md`: Team discussion log and assessment of suggestions.
- `08_BRAINSTORM_IDEAS.md`: Concrete feature ideas and experiments.

## Update Policy
- Keep this folder in version control.
- Update state and plans after every meaningful merge to `main`.
- Treat this as operational documentation, not marketing documentation.
